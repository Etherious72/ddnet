# 从 0 训练到测试（逐步操作）

本指南基于当前仓库的“手动参数配置”模式编写，不使用命令行参数。

## 1. 环境准备

1) 进入项目根目录：

```bash
cd D:\coding\LU\ddnet-main
```

2) 确认 Python 可用：

```bash
python --version
```

3) 确认依赖可用：

```bash
python -c "import torch, numpy, scipy, cv2, matplotlib"
```

如报错，请先在你的环境中安装依赖（PyTorch/NumPy/SciPy/OpenCV/Matplotlib）。

## 2. 路径与数据

1) 打开 `path_config.py`，确保 `main_dir` 指向当前项目根目录。

2) 确认数据目录结构（以 OpenFWI 为例）：

```
data/CurveFaultA/train_data/seismic/seismic*.npy
data/CurveFaultA/train_data/vmodel/vmodel*.npy
data/CurveFaultA/test_data/seismic/seismic*.npy
data/CurveFaultA/test_data/vmodel/vmodel*.npy
```

## 3. 训练参数设置

打开 `param_config.py`，至少确认以下参数：

- `dataset_name`（如 `CurveFaultA`）
- `model_type`（如 `DDNet70`）
- `device_mode`（`auto|cpu|gpu`）
- `train_size` / `test_size`
- `train_batch_size` / `test_batch_size`
- 学习率与各阶段 epoch

## 4. 监督训练（主训练）

1) 打开 `model_train.py`，设置 `TRAIN_MANUAL_CONFIG`：

- `model_type`
- `load_pretrained`（留空表示从头训练）
- 说明：本次实现采用隐式模式来区分 finetune 与 from-scratch，输出目录会基于模式自动切换。
- `finetune_lr_scale`

2) 运行训练：

```bash
python model_train.py
```

训练完成后，输出将落在分模式的根输出目录中，方便对比与复现实验。具体分布如下：
- finetune（已使用预训练权重继续训练）对应的输出根目录：`results/used_pretrain/`。
- from-scratch（从头训练）对应的输出根目录：`results/unused_pretrain/`。

输出目录结构示例：
- `results/used_pretrain/<model_type>/<dataset>/<run_id>/`
- `results/unused_pretrain/<model_type>/<dataset>/<run_id>/`

说明：同一训练任务在不同阶段会使用同一个 `run_id`，以便将阶段产出聚合到同一个实验下；并在该根目录下记录元数据（如 `run_meta.json`），方便回溯与复现实验。

实际保存的内容包括：模型权重、阶段性检查点、训练日志以及损失曲线等，全部落在对应的 run_id 子目录中。

```
models/<dataset_name>Model/
```

## 5. 测试（推荐：多模型对比）

1) 打开 `compare_models.pretrain.json`，设置模型路径：

- `models[0].model_path`：A 组模型路径
- `models[1].model_path`：B 组模型路径（如无 B 组可先删除这一项）
- `sort_by` 建议设为 `mse_mean`

2) 打开 `model_test.py` 底部，设置：

```python
TEST_MANUAL_CONFIG = {
    "compare_config": "compare_models.pretrain.json",
    "enable_lpips": False,
}
```

3) 运行测试：

```bash
python model_test.py
```

输出结果在：

```
results/<dataset_name>Results/[CompareBatch]...csv
results/<dataset_name>Results/[CompareSingle]...csv
```

## 6. 自监督预训练与微调（可选）

### 6.1 自监督预训练

1) 打开 `selfsup_msae_pretrain.py`，设置 `MANUAL_CONFIG`。
2) 运行：

```bash
python selfsup_msae_pretrain.py
```

输出权重在：

```
models_pretrain/<target_tag>Model/*.pth
```

### 6.2 自监督权重微调

1) 打开 `selfsup_to_ddnet70_finetune.py`，设置：

- `pretrained_path` 指向上一步 `.pth`
- `models_root = "models_selfsup"`

2) 运行：

```bash
python selfsup_to_ddnet70_finetune.py
```

输出模型在：

```
models_selfsup/<dataset_name>Model/
```

## 7. 常见问题

- 若出现窗口或 Tk 相关崩溃：确保 `model_test.py` 中 `enable_lpips=False` 且 matplotlib 后端为 `Agg`。
- 若模型文件找不到：检查 `compare_models.pretrain.json` 的路径是否正确。
